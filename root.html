O que é root?


Root é o processo de obter acesso de superusuário em um dispositivo Android, permitindo modificações avançadas no sistema como costumzações e configurações indisponiveis do seu android.

O que é o magisk, Apatch ou KernelSU?


O Magisk é uma ferramenta que permite gerenciar o acesso root de forma mais segura, permitindo até instalação de extensões/módulos. Ele também pode ocultar o acesso root para aplicativos específicos que não aceitam o root por conta das diretizes do APP.
O Apatch e um gerênciamento de root aonde dá para colocar módulos e gerenciar os apps que precisam de root. O Apatch usa o método de senha, aonde precisa de uma senha para poder usar o root como instalar módulo e etc.
O Apatch Lite e uma versão modificada aonde tira as partes negativas do Apatch normal como a própria senha para poder usar o root, entretanto ele não está funcionando em todos os celulares perfeitamente.
O KerenlSU é um gerenciamento de root aonde é colocado na própria source Kernel do device, Ele é gerenciamento de root mais atual aonde tem mais segurança, ele é tambem o mais escondido de todos (quer dizer que ele pode ser escondido em apps de bancos e jogos).
